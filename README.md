#  Proyecto E-commerce - Capibara Store

Este proyecto corresponde al desarrollo del sitio de e-commerce
## Funcionalidades principales
- Barra de navegación 
- Listado de productos en tarjetas con imagen, nombre y precio.
- Botones para comprar y agregar al carrito.
- Página de descripción detallada de cada producto.
- Footer con información de la tienda.

## Tecnologías utilizadas
- **HTML5**
- **CSS**
- **Bootstrap**
- **JavaScript**
- **Git/GitHub**

## Estructura del proyecto
- index.html  Página principal con listado de productos.
- producto.html  Página de descripción del producto.
- css/  Estilos personalizados.
- js/  Funciones de interacción.
- img/  Imágenes de productos.
